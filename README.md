## Installation
```
# Install Kitty terminal (if not installed)
$ brew install kitty

# Install language servers (typescript, diagnostic), eslint and prettier globally:
$ npm i -g typescript typescript-language-server diagnostic-languageserver eslint_d prettier_d_slim prettier

# Install NNN file manager
$ brew install nnn

# Install Fish shell
$ brew install fish

# Oh My Fish
$ curl https://raw.githubusercontent.com/oh-my-fish/oh-my-fish/master/bin/install | fish
```
